After having trouble with the collision system between a moving circle and a moving rectangle I decided that it was a good opportunity to learn how to use a physics engine. In this case im using Matter.js made by liabru. The main reasons are that it has a pretty good documentation and community. Also I decided to add some sounds and to use a custom font to learn how to do different things. The font is press start 2p made by Cody "CodeMan38" Boisclair (cody@zone38.net)

Player 1 controls the right paddle with the up and down arrows.

In the 2 players mode Player 2 controls the left paddle with the W and S keys.
